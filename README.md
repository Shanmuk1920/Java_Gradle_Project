# Java Gradle Application

## Project Overview
This project is a simple Java application built using Gradle as the build automation tool. It demonstrates how to create, build, and run a Java project using Gradle.

## Technologies Used
- Java
- Gradle
- Git
- GitHub

## Project Structure
app/
 ├── src/
 │   ├── main/java/       # Application source code
 │   └── test/java/       # Test files
 ├── build.gradle
 └── settings.gradle

## Requirements
Make sure the following are installed:

- Java (JDK 17 or above)
- Gradle

## Build the Project

Run the following command in the project directory:

gradle build

This will compile the code and create the build files.

## Run the Application

To run the application:

gradle run

## Features
- Simple Java application
- Built using Gradle build tool
- Demonstrates project initialization and build process

## Learning Outcomes
- Understanding Gradle build tool
- Java project structure
- Build and run Java applications using Gradle
